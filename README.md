

# 5U 战神

Small-margin, high-leverage BTC playground with real order tracking and honor system.

## Run Locally

**Prerequisites:** Node.js + pnpm

1. Install dependencies: `pnpm install`
2. The app will prompt to add/switch to a signing network named `HL Signer` (chainId 1337) for Hyperliquid order signatures.
4. Start dev server: `pnpm dev`
